#### 注意事项

<details markdown='1'><summary>展开/收起</summary>

>打开呆毛版脚本的命令
```
xdm
```
>获取呆毛版脚本帮助的命令
```
xdm help
```
>修复呆毛版脚本打不开的命令
```
bash <(curl -sL https://gitee.com/Misaka21011/Yunzai-Bot-Shell/raw/master/install.sh)
```
>删除呆毛版脚本的命令
```
rm /usr/local/bin/xdm
```

</details>
