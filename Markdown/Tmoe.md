# 安装云崽

- 使用 `xdm` 命令打开呆毛版脚本
- 选择 `TRSS-Yunzai` 并 `回车` 来安装TRSS崽
- 返回主页，选择 `NapCat` 安装、配置接口后启动 `NapCat` 
- 首次启动使用前台启动 `TRSS-Yunzai` ，成功连接上bot之后发送 `#设置主人`，待稳定运行后按 `ctrl+c` 退出前台，从 `xdm` 中使用后台启动。（前台启动不能关闭控制台，否则机器人会关闭，所以稳定运行后，改用后台启动）
- 在后台正常启动后，可以选择自己喜欢的插件安装啦~
